# pidicon2025

2025 가명정보 활용 경진대회 참가 프로젝트 (흑석동 물주먹)

## 프로젝트 구성

- 정형데이터 처리
- 비정형데이터 처리 (음성 데이터)
- 가명처리 기술 적용

## 파일 설명

| 파일명 | 설명 |
|--------|------|
| `anonymization.ipynb` | 가명처리 코드 |
| `정형데이터_처리문서.docx` | 정형데이터 처리 문서 |
| `비정형데이터_처리문서.docx` | 비정형데이터 처리 문서 |
| `사례1_가명처리.m4a` | 음성 가명처리 사례 1 |
| `사례2_가명처리.m4a` | 음성 가명처리 사례 2 |
| `사례3_가명처리.m4a` | 음성 가명처리 사례 3 |
| `흑석동물주먹_2025 가명정보활용경진대회_기술부문_본선_정형데이터.csv` | 정형데이터 결과물 |
